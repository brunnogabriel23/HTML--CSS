Principais Formatações

Negrito / destaque

Nesta frase tem um termo em negrito usando a tag <b> </b> (Não semântica)

Nesta frase tem um termo em destaque usando a tag <strong> </strong> (Semântica)

Itálico / Enfase

Nesta frase, temos um termo Itálico usando a tag <i> (Não semântica)

Nesta frase, temos um termo em ênfase usando a tag <em> (Semântica)

Texto marcado

Podemos também criar um texto marcado com a tag <mark> e </mark>

Segue o líder JOVEM PLAN

Estamos criando um texto grande e um texto pequeno Para criar um texto grande usamos a tag <big> e </big> e <small> e </small> para criar um texto pequeno.

Texto Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado. Para fazermos isso colocamos a tag <del> e </del>

Texto inserido

Podemos marcar um texto como inserido para dar uma ênfase e indicar que ele foi adicionado depois. Para fazermos isso colocamos a tag <ins> e </ins>

Existe também o sublinhado com a tag <u> e </u>

Texto sobrescrito

Para criar coisas do tipo x20+3 Para criar isso usamos a tag <sup> e </sup>

Texto subscrito

Para criar coisas do tipo H20 Para criar isso usamos a tag <sub> e </sub>
